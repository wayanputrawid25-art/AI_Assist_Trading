# 4D

4D Historical Pattern Analyzer - Laravel 12 + Vue 3 + PostgreSQL

## About

This is a web application for analyzing historical 4-digit patterns with statistical analysis, OCR support, and data visualization.

## Features

- Landing page with dark mode
- Dashboard with statistics
- Upload CSV/Excel/image files
- OCR with Tesseract integration
- Statistical analysis for AS, KOP, KEPALA, EKOR positions
- Hot/Cold digit analysis
- Digit frequency heatmaps
- Export to Excel, CSV, PDF

## Tech Stack

- Laravel 12
- Vue 3 + Vue Router + Vue Query
- Tailwind CSS
- PostgreSQL
- Chart.js
